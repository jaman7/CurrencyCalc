# sample CurrencyCalc React
Sample CurrencyCalc React/Mobx app.

## Installation:

## Use git clone.
```
git clone https://github.com/jaman7/CurrencyCalc.git
```
## install
```
yarn install
```

## run command

```
gulp
```
